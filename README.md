# 📄 C.V. Profesional – Manuel D. Labrador R.
 
Documentación del proceso de construcción de mi CV en formato HTML semántico.
 
---
 
## 🧠 ¿Por qué HTML para un CV?
 
Un CV en HTML permite:
 
- **Control total del diseño** sin depender de herramientas como Word o Canva.
- **Portabilidad:** se abre en cualquier navegador sin instalar nada.
- **Escalabilidad:** fácil de agregar CSS, convertirlo a PDF o desplegarlo en la web.
- **Práctica real:** aplicar conocimientos de HTML semántico en un proyecto propio.
 
---
 
## 🏗️ Proceso de Construcción
 
### 1. Definir el contenido
 
Antes de escribir una línea de código, organicé la información en secciones:
 
- Encabezado con datos de contacto
- Perfil profesional
- Experiencia laboral
- Habilidades técnicas
- Formación académica
- Idiomas y disponibilidad
 
### 2. Estructura HTML base
 
Comencé con el esqueleto estándar de cualquier documento HTML5:
 
```html
<!DOCTYPE html>
<html lang="es">
<head>
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<title>Valentina Paz Llantén Robles – CV Profesional 2026</title>
</head>
<body>
<!-- contenido -->
</body>
</html> ´´´
